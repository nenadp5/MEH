# MEH
Mehatronika
Ovde možete pronaći sadržaj koji vam je potreban za polaganje pismenog dela ispita iz predmeta "Mehatronika"
- U okviru fajla "Računske vežbe" možete pronaći tekst zadatka, zajedno sa kodom i prikazanom šemom koja se od vas traži.
- Što se tiče fajla "Blanketi", tu se nalaze blanketi iz različitih ispitnih rokova (stara i nova akreditacija). Blanket se uglavnom sastoji iz 4 ili 5 zadataka. Većina zadataka su slična kao iz predmeta "Algoritmi i programiranje", osim poslednjeg koji sadrži šemu koju je potrebno povezati.
- Potrudiću se da većina zadataka bude propraćena sa odgovarajućom šemom, ukoliko je to moguće realizovati u simulatoru "Tinkercad"

Ukoliko pronadjete bilo kakvu grešku, prijavite kako bi je uklonili
